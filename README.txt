App Description: This application allows people to make a list of their favorite activities so that
they can be reminded of all the fun things they like to do when they are bored.  If they get bored
and none of the ideas on their list appeal to them, the app will automatically generate new activities
for them to try.  

API Links: https://www.boredapi.com/
